if就是在程序用来做判断的

说明：if –> 是python的关键字
条件语句 –> 要求最终有一个值，可以是一个字面量，也可以是一个变量…一般情况下要求结果是个布尔值

执行语句块 –>可以是一行或者是多行代码。这个代码中的代码必须和前面中的if保存一个缩进

2.range函数

a.range(n): 产生一个0 ~ n-1范围对应整数序列

b.range(m,n): 产生一个m ~ n-1范围对应的整数序列

c.range(m,n,step): 产生一个m ~ n-1中间隔step范围对应的整数序列

练习：计算1+2+3+…+100

python中没有do-while
“””

while 条件语句:
循环体

说明：
while -> 关键字
条件语句 -> 和if后面的条件语句要求一样,可以是任何有值的表达式(不能是赋值语句)
: -> 固定写法
循环体 -> 重复执行的代码段。循环体要和while保持一个缩进

执行过程：判断条件语句的结果是否为True，为True执行循环体。
执行完循环体，再判断条件语句是否为True,为True又执行循环体…
依次类推，知道判断条件语句的结果为False，循环就结束

#####注意： while True: —-> 死循环

1.continue

只能写在循环体中，作用是，当遇到continue就结束当次循环直接进行下次循环的判断
下次循环的判断：for -> 让变量取下一个值 while -> 判断条件是否为True

练习：计算1~100中非奇数的和 (使用continue)

2.break

只能写在循环体中
只要在循环中遇到break,整个循环直接结束。执行循环后面的其他语句
